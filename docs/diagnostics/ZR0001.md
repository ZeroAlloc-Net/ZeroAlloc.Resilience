# ZR0001 — Fallback Not Found or Signature Mismatch

**Severity:** Error

A `[CircuitBreaker(Fallback = nameof(...))]` was declared but the named method could not be found on the interface, or its parameter types / return type do not match the annotated method.

---

## Examples

### Method not found

```csharp
// ZR0001: 'GetCached' does not exist on IExternalService
[CircuitBreaker(MaxFailures = 3, ResetMs = 1_000, HalfOpenProbes = 1, Fallback = "GetCached")]
public interface IExternalService
{
    ValueTask<string> GetAsync(string id, CancellationToken ct);
}
```

### Return type mismatch

```csharp
// ZR0001: GetFallback returns string but GetAsync returns ValueTask<string>
[CircuitBreaker(MaxFailures = 3, ResetMs = 1_000, HalfOpenProbes = 1, Fallback = nameof(GetFallback))]
public interface IExternalService
{
    ValueTask<string> GetAsync(string id, CancellationToken ct);
    string GetFallback(string id, CancellationToken ct); // ← wrong return type
}
```

### Parameter mismatch

```csharp
// ZR0001: GetFallback has a different parameter list
[CircuitBreaker(MaxFailures = 3, ResetMs = 1_000, HalfOpenProbes = 1, Fallback = nameof(GetFallback))]
public interface IExternalService
{
    ValueTask<string> GetAsync(string id, CancellationToken ct);
    ValueTask<string> GetFallback(CancellationToken ct); // ← missing 'id' parameter
}
```

---

## How to fix

1. Ensure the fallback method exists on the same interface.
2. Ensure the fallback has **exactly the same parameter types** in the same order.
3. Ensure the fallback has **exactly the same return type**.

Correct:

```csharp
[CircuitBreaker(MaxFailures = 3, ResetMs = 1_000, HalfOpenProbes = 1, Fallback = nameof(GetFallback))]
public interface IExternalService
{
    ValueTask<string> GetAsync(string id, CancellationToken ct);
    ValueTask<string> GetFallback(string id, CancellationToken ct); // ✓ matches
}
```

Use `nameof(...)` rather than a string literal to keep the reference refactor-safe:

```csharp
Fallback = nameof(IExternalService.GetFallback)
// or simply:
Fallback = nameof(GetFallback)
```
